### 开发环境MacOS(High Sierra 10.13.5)

#### 需要的信息和信息源：

* 域名需要备案，这是由国家法律规定的，从表明上看，这是为了维护好网络秩序，一旦发生扰乱秩序的情况，能够进行溯源和追究

* 目前（2018.08.23），域名备案的国际受理部门是 [工业和信息化部](http://www.miitbeian.gov.cn/)，换句话说“当你需要上线运营的那一刻，你就注定要和政府打交道了”

* 备案的流程，是受到政策影响的，目前的拍板权交由 [工业和信息化部](http://www.miitbeian.gov.cn/)

* 阿里云和腾讯云等类似的公司，他们属于运营商，没有拍板权

* 我们可以委托阿里云和腾讯云等公司协助备案，通过他们网站上提供的备案系统，可以加快备案的速度和成功率

* 这里我们先选择阿里云作为我们的服务提供商，如果对阿里云不满意，以后我们就换别家公司

* 我们之前在阿里云购买了域名，并绑定邮箱和实名认证过了，接下来我们需要在阿里云购买服务器进行备案

* 在阿里云ecs服务器区，挑选一个价格最低带有公网ip的服务器，可以先买一个月（或者有新人优惠的话，建议使用新人优惠进行购买）

* 心疼钱的话，就加把劲学会了技术作出项目再赚回来

* ecs服务器（其实就是个主机，跟你那电脑主机差不多，只不过人家有公网ip，而你没有；因为公网ip因为《ipv4协议》的原因的数量有限，所以ip地址事实上是被垄断了，当然了，为了提供和维护ip地址也是需要成本的）购买成功后，需要让你的域名和ip地址关联起来，这个过程就是域名解析了

* 域名正常解析的状态下，访问你的域名才能访问到你的ip地址-也就是你的服务器

* 域名的解析，每个服务提供商都是大同小异，并且都会有教程，照着他们的教程做就行了，如果有问题，就致电客服，在运维中，要善于通过各种途径和方法去解决问题；额，其实，人生每一件大大小小的事又何尝不是呢？

* 从浏览器访问一个域名的大概流程：

``` txt
浏览器输入域名“arthur.com” -> 浏览器在本地查找有没有跟“arthur.com”对应的ip地址：
1.有   -> 浏览器访问对应的ip地址
2.没有 -> 浏览器访问 根域名服务器（大机构掌控），根域名服务器搜索域名对应的ip地址，然后把结果返回给浏览器
      -> 浏览器访问对应的ip地址，同时可能把这对 域名<->ip地址 记录在你的电脑上
```

* 至此，希望通过理解和多联系实际情况，完成 域名的解析。
